# 자원 순환 시스템

## 배터리 시스템 (핵심 자원)

### 수급 방식

#### 상점 구매
- **가격**: 높게 책정 (경제적 압박 조성)
- **목적**: 기본 생존 보장
- **제한**: 돈이 있어야 구매 가능
- **전략적 의미**: 안정성 vs 효율성

#### 파밍 획득
- **위치**: 탐험 지역에서 발견
- **목적**: 추가 생존 시간 확보
- **리스크**: 안개 속 탐험 필요
- **전략적 의미**: 위험 감수 vs 보상

### 사용 구조

#### 탐험 중 사용
- **손전등**: 시야 확보 (안개 속 필수)
- **통신 장비**: 팀원과 소통
- **휴대 전자기기**: 제한적 CCTV 조작
- **기타 탐험 도구**

#### 방어 중 사용
- **센터 전력**: 기본 시설 운영
- **CCTV**: 감시 시스템
- **문 제어**: 방어 메커니즘
- **조명**: 시야 확보

## 기타 자원 카테고리

### 즉시 사용형

#### 🔥 화염 트랩 연료
- **불통**: 인간형 크리처 제거
- **구매처**: 상점
- **특징**: 높은 데미지, 느린 재장전
- **사용법**: 크리처 속성 확인 후 사용

#### 💧 물 트랩 연료
- **물통**: 기계형 크리처 제거
- **구매처**: 상점
- **특징**: 중간 데미지, 보통 재장전
- **사용법**: 렌페이지 등 기계형 대상

#### 🔨 물리 트랩 연료
- **유압가스통**: 노말형 크리처 제거
- **구매처**: 상점
- **특징**: 안정적 데미지, 빠른 재장전
- **사용법**: 범용성 높음

#### 기타 즉시 사용형
- **쿠션**: 렘페이지 돌진 차단용
- **바리케이드 재료**: 추가 방어 구조물
- **의료용품**: 플레이어 회복
- **수리 도구**: 센터 보수

### 장기 투자형

#### 경제적 가치
- **폐철물**: 판매용 스크랩
- **고가치 아이템**: 큰 돈 획득
- **특수 합금**: 업그레이드 재료

#### 시설 개선
- **업그레이드 재료**: 센터/장비 개선
- **효율성 부품**: 전력 소모 감소
- **확장 키트**: CCTV 추가 설치

#### 특수 아이템
- **크리처 기피제**: 일정 시간 안전 보장
- **강화 배터리**: 더 오래 지속되는 전력
- **통신 증폭기**: 더 먼 거리 팀원 연락

## 경제 시스템

### 상점 시스템

#### 기본 아이템
| 아이템 | 가격 | 효과 | 비고 |
|--------|------|------|------|
| 배터리 | 높음 | 전력 공급 | 생존 필수 |
| 불통 | 중간 | 인간형 제거 | 화염 트랩 |
| 물통 | 중간 | 기계형 제거 | 물 트랩 |
| 유압가스통 | 중간 | 노말형 제거 | 물리 트랩 |
| 쿠션 | 낮음 | 렘페이지 대응 | 일회용 |

#### 업그레이드 아이템
| 아이템 | 가격 | 효과 | 비고 |
|--------|------|------|------|
| CCTV 추가 | 매우 높음 | 감시 범위 확장 | 영구 효과 |
| 전력 효율화 | 높음 | 소모량 10% 감소 | 영구 효과 |
| 센터 강화 | 높음 | HP 20 증가 | 영구 효과 |

### 가격 밸런싱 철학

#### 경제적 압박 조성
- **배터리**: 비싸게 → 파밍 유도
- **트랩 연료**: 적당히 → 전략적 선택
- **업그레이드**: 매우 비싸게 → 장기 목표

#### 위험-보상 균형
```
안전 지역 아이템 가치: 1.0x
위험 지역 아이템 가치: 1.5x
극위험 지역 아이템 가치: 2.0x
```

## 자원 관리 시스템

### 인벤토리 제약

#### 슬롯 기반 시스템
- **개인 인벤토리**: 8슬롯 (예시)
- **무게 제한**: 없음 (슬롯만 제한)
- **아이템 스택**: 같은 종류 최대 5개

#### 전략적 선택 강요
- **배터리 vs 트랩 연료**: 생존 vs 공격
- **즉시 사용 vs 장기 투자**: 단기 안전 vs 장기 이익
- **개인 vs 팀 자원**: 나눠 가질지 한 명이 들고 다닐지

### 팀 자원 공유

#### 센터 저장소
- **공용 창고**: 8슬롯 (팀 공유)
- **접근 조건**: 센터 내부에서만
- **용도**: 다음 사이클 준비물 보관

#### 실시간 공유
- **아이템 건네주기**: 근접 시 가능
- **긴급 지원**: 배터리 나눠주기
- **역할 분담**: 트랩 담당자에게 연료 집중

## 자원 순환 메커니즘

### 사이클별 자원 흐름

#### 탐험 페이즈
```
탐험 시작 → 아이템 수집 → 위험도 증가 → 선택의 딜레마
├── 더 탐험: 고위험 고보상
└── 안전 복귀: 저위험 저보상
```

#### 방어 페이즈
```
자원 소모 시작 → 전략적 사용 → 위기 관리 → 생존 or 실패
├── 성공: 다음 사이클 진행
└── 실패: 게임오버
```

### 자원 압박 설계

#### 점진적 압박 증가
- **사이클 1-3**: 여유로운 자원
- **사이클 4-7**: 균형잡힌 압박
- **사이클 8+**: 지속적인 자원 부족

#### 선택의 딜레마 유발
- **탐험 vs 안전**: "더 나가서 아이템을 찾을까?"
- **현재 vs 미래**: "지금 쓸까, 나중을 위해 아낄까?"
- **개인 vs 팀**: "내가 가져야 하나, 팀원에게 줘야 하나?"

## 밸런싱 고려사항

### 자원 부족 vs 풍족

#### 부족한 상황 (의도된 압박)
- **긴장감 조성**: 매 선택이 중요해짐
- **전략적 사고**: 효율적 사용 필요
- **팀워크 강화**: 자원 공유 필요

#### 풍족한 상황 (피해야 할 상황)
- **긴장감 상실**: 실수해도 여유있음
- **전략 무의미**: 아무거나 써도 됨
- **팀워크 불필요**: 개인 플레이 가능

### 균형점 찾기

#### 목표 밸런스
- **생존 가능**: 기본적인 생존은 보장
- **선택 중요**: 잘못된 선택 시 위험
- **팀워크 필수**: 혼자서는 한계

#### 측정 지표
- **생존률**: 사이클별 50-70% 유지
- **자원 사용률**: 80-90% 활용
- **팀 의존도**: 솔로 vs 팀 성공률 차이 30%+

## 추가 자원 아이디어 (확장 시 고려)

### 새로운 자원 타입
- **에너지 셀**: 배터리보다 고효율
- **방어막 생성기**: 일시적 무적
- **시간 조작 장치**: 방어 페이즈 시간 연장
- **크리처 분석기**: 속성 자동 식별

### 자원 조합 시스템
- **폐철물 + 배터리 = 강화 배터리**
- **의료용품 + 특수 화학물질 = 회복 부스터**
- **전자부품 + 유압가스 = 자동 트랩**

### 지역별 특수 자원
- **의료실**: 고급 의료용품
- **전기실**: 고효율 배터리
- **연구실**: 크리처 관련 정보
- **창고**: 대량 기본 자원
